Herman Marshall

Concept

The Herman Marshall website is a place where users can find information about the brand, the founders, where they can buy the whiskey, and what kind of mixed drink they can create with it.

Design process

While creating a new website for Herman Marshall, I looked at the old website and created a mood board. The website gave off a nice and homely vibe, something I that greatly inspired me. After creating the mood boards, I started to create sketches, however I soon moved on to design in Adobe Xd.

The biggest challenges in this design were, recreating the homely feeling while simultaneously having a well-structured readable website that can be used different user groups.
